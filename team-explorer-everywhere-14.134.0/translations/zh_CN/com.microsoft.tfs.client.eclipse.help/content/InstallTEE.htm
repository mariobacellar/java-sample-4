正在安装 Team Explorer Everywhere 插件和命令行客户端

你可以在借助 Eclipse 进行开发的过程中使用 Team Explorer Everywhere 和 Visual Studio Team Foundation Server 来跟踪工作项、提供版本控制和自动处理生成。在使用这些功能之前，你必须安装 Team Explorer Everywhere 面向 Eclipse 的插件或面向 Team Foundation Server 的跨平台命令行客户端。

本主题内容

  • 系统必备
  • 安装 Team Explorer Everywhere 面向 Eclipse 的插件
  • 安装 Team Foundation Server 的跨平台命令行客户端

系统必备

您必须安装受支持的 Eclipse 版本，可以从以下网站获取该版本： Eclipse.org 主页.

操作系统要求：

  • Windows 10 (x86 和 x64)
  • Windows 8.1 (x86 和 x64)
  • Windows 8 (x86、x64)
  • Windows 7 (x86、x64)
  • Linux with GLIBC 2.3+ (x86、x86_64)
  • Mac OS X 10.8+ (仅 Intel)

集成开发环境 (IDE) 要求:

  • Linux、Mac OS X 或 Windows 上，基于 Eclipse 4.2+ 的 IDE。

Java 运行时要求:

  • Linux 上，Oracle Java™ 1.7+ 或 IBM Java™ 1.7+
  • Mac OS X 上，Apple Java™ 1.7+
  • Windows 上，Oracle Java™ 1.7+ 或 IBM Java™ 1.7+

Team Foundation Server 要求:

  • Visual Studio Team Services
  • Visual Studio Team Foundation Server 2017
  • Visual Studio Team Foundation Server 2015
  • Visual Studio Team Foundation Server 2013
  • Visual Studio Team Foundation Server 2012

安装 Team Explorer Everywhere 面向 Eclipse 的插件

你必须先安装插件，然后才可在开发过程中使用该插件。 可直接从更新站点安装到 Eclipse 中，也可通过 Eclipse 更新站点存档安装。

从更新站点安装插件

  1. 打开 Eclipse。
  2. 在“帮助”菜单上，选择“安装新软件”。
  3. 将显示“安装”对话框。
  4. 选择“添加”按钮。
  5. 将显示“添加存储库”对话框。
  6. 相关名称，请输入 "Team Explorer Everywhere"
  7. 指定更新站点的位置: http://dl.microsoft.com/eclipse/tfs
  8. 选择“确定”按钮。
  9. 在“安装”对话框的功能列表中，选择与 Team Explorer Everywhere 面向 Eclipse 的插件对应的复选框。如果你未看到此选项，请使用辅助的下拉菜单，找到刚在列表中输入的更新站点 URL 并将其选中，然后选择上述所提插件旁边的复选框。
  10. 选择“下一步”按钮两次。
  11. 如果你同意 Microsoft 软件许可条款，请接受这些条款，然后选择“完成”按钮。
  12. 在出现提示时重启 Eclipse。

从更新站点存档安装插件

    注意
    上述标准更新站点机制是首选安装方法，但这需要连接到 Internet，并将安装 Microsoft 提供的最新版本的插件。 如果安装计算机未直接连接到 Internet，或者如果组织希望每个人安装特定版本的插件，则以下方法可能更合适。

  1. 使用此页上的链接下载 TFSEclipsePlugin-UpdateSiteArchive-14.0.4.zip。
  2. 打开 Eclipse。
  3. 在“帮助”菜单上，选择“安装新软件”。
  4. 将显示“安装”对话框。
  5. 选择“添加”按钮。
  6. 将显示“添加存储库”对话框。
  7. 相关名称，请输入“本地 Team Explorer Everywhere 更新档案”。
  8. 选择“存档”按钮。
  9. 将显示“存储库归档”对话框。
  10. 查找并选择下载的“TFSEclipsePlugin-UpdateSiteArchive-14.0.4.zip”文件作为文件名。
  11. 选择“打开”按钮。
  12. 选择“确定”按钮。
  13. 在“安装”对话框的功能列表中，选择与 Team Explorer Everywhere 面向 Eclipse 的插件对应的复选框
  14. 选择“下一步”按钮两次。
  15. 如果你同意 Microsoft 软件许可条款，请接受这些条款，然后选择“完成”按钮。
  16. 在出现提示时重启 Eclipse。

安装适用于 Team Foundation Server 的跨平台命令行客户端

可从适用于 Team Foundation Server 的跨平台命令行客户端执行多种版本控制操作。 必须先安装此客户端，并将 shell 或系统路径配置为包括安装文件夹，然后才可使用此客户端。

安装适用于 Team Foundation Server 的跨平台命令行客户端

  1. 使用此页上的链接下载 TEE-CLC-14.0.4.zip。
  2. 解压缩包含客户端的存档(TEE-CLC-14.0.4.zip 文件)。
  3. 将 shell 或系统路径配置为包括存档解压缩到的文件夹。
  4. 要验证客户端是否正在工作，请在命令提示符处键入 tf，然后按 ENTER 键。
如果正确安装了客户端，则将显示输出，以“Microsoft Team Explorer Everywhere 命令行客户端(version 14.0.4)”开头。

有关任何命令的信息，可键入 tf help Command，其中 Command 是你要了解其信息的命令的名称。 例如，你可以键入 tf help checkin。 也可键入 tf command /help 或 tf command -help 以获取帮助。

注意
可通过在 shell 中设置环境变量(例如，指定要使用的 HTTP 代理)控制命令行客户端的行为，还可启用凭据的自动缓存或控制差异/合并工具。 有关要设置的环境变量的信息，请参阅 设置环境变量 MSDN 上的主题。
