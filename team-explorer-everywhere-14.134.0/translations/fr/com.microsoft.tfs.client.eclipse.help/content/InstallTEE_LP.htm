Installation du module linguistique du plug-in Team Explorer Everywhere et du client en ligne de commande

Vous pouvez utiliser le module linguistique Team Explorer Everywhere pour ajouter la prise en charge des langues suivantes dans le plug-in Team Explorer Everywhere et dans le client en ligne de commande.

  • Portugais (Brésil)
  • Chinois (simplifié)
  • Chinois (traditionnel)
  • Français
  • Allemand
  • Japonais

Composants requis

Vous devez avoir installé Team Explorer Everywhere et la version de Team Foundation Server dans la langue souhaitée pour pouvoir utiliser ce module linguistique.

Ajout de langues supplémentaires à Team Explorer Everywhere

Vous pouvez installer le plug-in du module linguistique pour afficher les messages et les fenêtres relatifs à Eclipse dans la langue de votre choix, à condition que celle-ci soit prise en charge par le module linguistique. Si un message spécifique n'est pas disponible dans la langue choisie, sa version anglaise s'affiche par défaut.

Remarque
Vous devez installer le module linguistique du produit Eclipse correspondant à votre langue et configurer Eclipse de manière à ce qu'il l'utilise. Consultez http://www.eclipse.org/babel/ pour plus d'informations sur le téléchargement et la configuration des modules linguistiques pour Eclipse.

Pour installer les langues supplémentaires de Team Explorer Everywhere dans Eclipse

  1. Ouvrez Eclipse.
  2. Dans le menu Help [Aide], cliquez sur Install New Software [Installer un nouveau logiciel]. La boîte de dialogue Install [Installer] s'affiche.
  3. Cliquez sur Add [Ajouter]. La boîte de dialogue Add Site [Ajouter un site] s'affiche.
  4. Dans le champ Name [Nom], tapez Local TFS Language Pack archive.
  5. Cliquez sur Archive [Archiver].
  6. Indiquez l'emplacement du fichier d'archive et cliquez sur Open [Ouvrir].
    Sélectionnez le fichier d'archive téléchargé TFSEclipsePlugin-NL1-UpdateSiteArchive-14.0.4.zip.
  7. Cliquez sur OK.
  8. Dans la liste des fonctionnalités de la boîte de dialogue Installer, cochez la case correspondant au module linguistique Team Explorer Everywhere.
  9. Cliquez deux fois sur Next [Suivant].
  10. Acceptez les Termes du contrat de licence logiciel Microsoft, puis cliquez sur Finish [Terminer].
  11. Redémarrez Eclipse lorsque vous y êtes invité.

Ajout de langues supplémentaires à Cross-platform Command-line Client for Team Foundation Server

Vous pouvez installer le module linguistique pour assurer la prise en charge de langues supplémentaires par Cross-platform Command-line Client for Team Foundation Server Les variables d'environnement spécifiques aux paramètres régionaux doivent être configurées pour la langue et l'encodage de caractères de votre choix. De plus, le programme de votre terminal doit être configuré de manière à afficher les caractères dans cet encodage. Pour plus d'informations, consultez la page du manuel des paramètres régionaux(1), ainsi que la documentation de votre programme de terminal.

Pour installer les langues supplémentaires du client de ligne de commande interplateforme

  1. Décompressez le fichier d'archive (TEE-CLC-NL1-14.0.4.zip) contenant les langues au même emplacement que le client de ligne de commande interplateforme.

Remarque
Le fichier d'archive contient un fichier com.microsoft.tfs.client.clc.nl1.jar unique qui est extrait dans le répertoire /lib de l'installation du client de ligne de commande. Confirmez que ce fichier figure dans le répertoire /lib de vos installations pour vérifier que l'installation a réussi.
