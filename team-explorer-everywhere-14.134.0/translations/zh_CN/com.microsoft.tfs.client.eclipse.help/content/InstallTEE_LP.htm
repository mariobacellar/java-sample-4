正在安装 Team Explorer Everywhere 插件和命令行客户端语言包

你可以使用 Team Explorer Everywhere 语言包在 Team Explorer Everywhere 插件和命令行客户端中添加对以下语言的支持。

  • 葡萄牙语（巴西）
  • 中文(简体)
  • 中文(繁体)
  • 法语
  • 德语
  • 日语

系统必备

您必须已安装 Team Explorer Everywhere 和所需语言版本的 Team Foundation Server 才能使用此语言包。

正在向 Team Explorer Everywhere 添加其他语言

您可以安装语言包插件来使用选定的 Eclipse 语言显示消息和窗口（如果受语言包的支持）。 如果无法用选定的语言显示特定消息，则将显示默认的英语版本。

注意
您应为您的语言安装 Eclipse 产品语言包，并将 Eclipse 配置为使用该语言包。 有关如何下载和配置 Eclipse 产品语言包的更多信息，请参见 http://www.eclipse.org/babel/ 。

将 Team Explorer Everywhere 附加语言安装到 Eclipse 中

  1. 打开 Eclipse。
  2. 在“帮助”菜单上，单击“安装新软件”。“ 安装”对话框随即出现。
  3. 单击“添加”。“ 添加网站”对话框随即出现。
  4. 在“名称”中，键入“本地 TFS 语言包存档”。
  5. 单击“存档”。
  6. 指定该存档文件的位置，然后单击“打开”。
    选择下载的存档文件 TFSEclipsePlugin-NL1-UpdateSiteArchive-14.0.4.zip。
  7. 单击“确定”。
  8. 在“安装”对话框的功能列表中，选择与 Team Explorer Everywhere 语言包对应的复选框。
  9. 单击“下一步”两次。
  10. 接受 Microsoft 软件许可条款，然后单击“完成”。
  11. 在系统提示时重新启动 Eclipse。

向 Team Foundation Server 的跨平台命令行客户端添加其他语言

可以安装此语言包以向 Team Foundation Server 的跨平台命令行客户端提供其他语言支持。 必须为所需语言和字符编码配置区域设置特定的环境变量，并且必须将终端程序配置为显示此编码中的字符。 有关更多信息，请参见终端程序的区域设置 (1) 手册页和文档。

安装跨平台命令行客户端附加语言

  1. 将包含语言的存档文件(TEE-CLC-NL1-14.0.4.zip)解压缩到跨平台命令行客户端所在的相同位置。

注意
此存档文件包含一个被提取到命令行客户端安装中的 /lib 目录的 com.microsoft.tfs.client.clc.nl1.jar 文件。 确认此文件位于安装 /lib 目录中以确定安装成功。
