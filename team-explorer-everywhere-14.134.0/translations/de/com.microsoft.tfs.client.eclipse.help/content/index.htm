Team Explorer Everywhere für Visual Studio Team Foundation Server

Team Explorer Everywhere in Kombination mit Microsoft Team Foundation Server erlaubt es Ihnen, die Zuverlässigkeit Ihrer Entwicklungsprozesse zu steigern. Team Explorer Everywhere enthält das Plug-In für Eclipse und den plattformübergreifenden Befehlszeilenclient für Team Foundation Server. Weitere Informationen zum Verwenden dieser Produkte finden Sie im Onlineleitfaden. Hilfe und zusätzliche Informationen finden Sie außerdem in den MSDN-Foren.
