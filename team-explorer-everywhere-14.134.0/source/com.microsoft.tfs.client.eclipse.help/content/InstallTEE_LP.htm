Installing the Team Explorer Everywhere Plug-in and the Command-line Client Language Pack

You can use the Team Explorer Everywhere Language Pack to add support for the following languages in the Team Explorer Everywhere Plug-in and the Command-line Client.

  • Portuguese (Brazil)
  • Chinese (Simplified)
  • Chinese (Traditional)
  • French
  • German
  • Japanese

Prerequisites

You must have already installed Team Explorer Everywhere and the desired language version of Team Foundation Server to use this language pack.

Adding Additional Languages to Team Explorer Everywhere

You can install the language pack plug-in to display messages and windows using the chosen Eclipse language if supported by the language pack. If a particular message is not available for the chosen language then the default English version will be displayed.

Note
You should install the Eclipse product language pack for your language and configure Eclipse to use it. See http://www.eclipse.org/babel/ for more information on how to download and configure product language packs for Eclipse.

To install the Team Explorer Everywhere additional languages into Eclipse

  1. Open Eclipse.
  2. On the Help menu, click Install New Software. The Install dialog box appears.
  3. Click Add. The Add Site dialog box appears.
  4. In Name, type Local TFS Language Pack archive.
  5. Click Archive.
  6. Specify the location of the archive file and click Open.
    Selected the downloaded archive file TFSEclipsePlugin-NL1-UpdateSiteArchive-14.134.0.zip.
  7. Click OK.
  8. In the list of features in the Install dialog box, select the check box that corresponds to the Team Explorer Everywhere Language Pack.
  9. Click Next two times.
  10. Accept the Microsoft Software License Terms, and then click Finish.
  11. Restart Eclipse when prompted.

Adding Additional Languages to the Cross-platform Command-line Client for Team Foundation Server

You can install the language pack to provide additional language support to the cross-platform command-line client for Team Foundation Server. Locale-specific environment variables must be configured for your desired language and character encoding, and your terminal program must be configured to display characters in this encoding. See the locale(1) manual page and the documentation for your terminal program for more information.

To install the Cross-Platform Command-Line Client additional languages

  1. Unzip the archive file (TEE-CLC-NL1-14.134.0.zip) that contains the languages into the same location as the cross-platform command-line client.

Note
The archive file contains a single com.microsoft.tfs.client.clc.nl1.jar file that is extracted to the /lib directory inside the command-line client installation. Confirm that this file is in your installations /lib directory to verify that the installation has been successful.
