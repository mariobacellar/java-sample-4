Das Team Explorer Everywhere-Plug-In und das Sprachpaket des Befehlszeilenclients werden installiert

Sie können das Team Explorer Everywhere-Sprachpaket verwenden, um Unterstützung für die folgenden Sprachen im Team Explorer Everywhere-Plug-In und im Befehlszeilenclient hinzuzufügen:

  • Portugiesisch (Brasilien)
  • Chinesisch (vereinfacht)
  • Chinesisch (traditionell)
  • Französisch
  • Deutsch
  • Japanisch

Erforderliche Komponenten

Team Explorer Everywhere und die gewünschte Sprachversion von Team Foundation Server müssen bereits installiert sein, um dieses Language Pack zu verwenden.

Team Explorer Everywhere werden weitere Sprachen hinzugefügt

Sie können das Language Pack-Plug-In installieren, um Meldungen und Fenster in der für Eclipse ausgewählten Sprache anzuzeigen, sofern diese vom Language Pack unterstützt wird. Wenn eine bestimmte Meldung in der ausgewählten Sprache nicht verfügbar ist, wird standardmäßig die englische Version angezeigt.

Hinweis
Sie müssen das Language Pack für das Eclipse-Produkt für Ihre Sprache installieren und Eclipse entsprechend konfigurieren. Weitere Informationen zum Herunterladen und Konfigurieren von Language Packs für das Eclipse-Produkt finden Sie unter http://www.eclipse.org/babel/ .

So installieren Sie zusätzliche Sprachen für Team Explorer Everywhere in Eclipse

  1. Öffnen Sie Eclipse.
  2. Klicken Sie im Menü Hilfe auf Neue Software installieren. Das Dialogfeld Installieren wird angezeigt.
  3. Klicken Sie auf Hinzufügen. Das Dialogfeld Site hinzufügen wird angezeigt.
  4. Geben Sie in Name den Eintrag Lokales Team Explorer Everywhere Language Pack-Archiv ein.
  5. Klicken Sie auf Archiv.
  6. Geben Sie den Speicherort der Archivdatei an, und klicken Sie auf Öffnen.
    Wählen Sie die heruntergeladene Archivdatei "TFSEclipsePlugin-NL1-UpdateSiteArchive-14.0.4.zip" aus.
  7. Klicken Sie auf OK.
  8. Aktivieren Sie in der Liste der Features im Dialogfeld Installieren das Kontrollkästchen, das dem Team Explorer Everywhere-Sprachpaket entspricht.
  9. Klicken Sie zweimal auf Weiter.
  10. Akzeptieren Sie die Microsoft-Softwarelizenzbedingungen, und klicken Sie dann auf Fertig stellen.
  11. Starten Sie Eclipse neu, wenn Sie dazu aufgefordert werden.

Hinzufügen zusätzlicher Sprachen zum plattformübergreifenden Befehlszeilenclient für Team Foundation Server

Sie können das Language Pack installieren, um im plattformübergreifenden Befehlszeilenclient für Team Foundation Server Unterstützung für zusätzliche Sprachen bereitzustellen. Gebietsschemaspezifische Umgebungsvariablen müssen für die gewünschte Sprache und Zeichencodierung konfiguriert werden. Außerdem müssen Sie das Terminalprogramm für die Anzeige von Zeichen in dieser Codierung konfigurieren.

So installieren Sie zusätzliche Sprachen für den plattformübergreifenden Befehlszeilenclient

  1. Entpacken Sie die Archivdatei (TEE-CLC-NL1-14.0.4.zip) mit den Sprachen in den Speicherort des plattformübergreifenden Befehlszeilenclients.

Hinweis
Die Archivdatei enthält eine einzelne Datei "com.microsoft.tfs.client.clc.nl1.jar", die in das Verzeichnis /lib innerhalb der Installation des Befehlszeilenclients extrahiert wird.
