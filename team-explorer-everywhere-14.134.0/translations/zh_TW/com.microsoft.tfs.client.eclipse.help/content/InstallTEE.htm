正在安裝 Team Explorer Everywhere 外掛程式與命令列用戶端

利用 Eclipse 可於開發程序中使用 Team Explorer Everywhere 與 Visual Studio Team Foundation Server，來追蹤工作項目、提供版本控制，以及將建置作業自動化。必須先安裝 Eclipse 的 Team Explorer Everywhere 外掛程式或 Team Foundation Server 的跨平台命令列用戶端，才可使用這些功能。

本主題內容

  • 必要條件
  • 安裝 Eclipse 的 Team Explorer Everywhere 外掛程式
  • 安裝適用於 Team Foundation Server 的跨平台命令列用戶端

必要條件

您必須安裝支援的 Eclipse 版本，請從下列網站取得支援的版本： Eclipse.org 首頁。

作業系統需求：

  • Windows 10 (x86 及 x64)
  • Windows 8.1 (x86 及 x64)
  • Windows 8 (x86、x64)
  • Windows 7 (x86、x64)
  • 包含 GLIBC 2.3+ 的 Linux (x86、x86_64)
  • Mac OS X 10.8+ (僅限 Intel)

整合開發環境 (IDE) 需求:

  • Linux、Mac OS X 或 Windows 上採用 Eclipse 4.2+ 的 IDE。

Java 執行階段需求:

  • Linux 上的 Oracle Java™ 1.7+ 或 IBM Java™ 1.7+
  • Mac OS X 上的 Apple Java™ 1.7+
  • Windows 上的 Oracle Java™ 1.7+ 或 IBM Java™ 1.7+

Team Foundation Server 需求:

  • Visual Studio Team Services
  • Visual Studio Team Foundation Server 2017
  • Visual Studio Team Foundation Server 2015
  • Visual Studio Team Foundation Server 2013
  • Visual Studio Team Foundation Server 2012

安裝 Eclipse 的 Team Explorer Everywhere 外掛程式

您必須安裝外掛程式，才可在開發處理序中使用。 您可以直接從更新網站安裝到 Eclipse，或從 Eclipse 更新網站封存安裝。

從更新網站安裝外掛程式

  1. 開啟 Eclipse。
  2. 在 [說明] 功能表上，選擇 [安裝新軟體]。
  3. [安裝] 對話方塊會隨即出現。
  4. 選擇 [新增] 按鈕。
  5. [新增儲存機制] 對話方塊會隨即出現。
  6. 請為名稱輸入 [Team Explorer Everywhere]
  7. 指定更新網站的位置: http://dl.microsoft.com/eclipse/tfs
  8. 選擇 [確定] 按鈕。
  9. 在 [安裝] 對話方塊的功能清單中，選取與 [Eclipse 的 Team Explorer Everywhere 外掛程式] 相對應的核取方塊。若沒有看到此選項，請利用 [使用] 的下拉功能表，在清單中找到剛剛所輸入的更新網站 URL 並加以選取，然後再選取上述外掛程式旁的核取方塊。
  10. 選擇 [下一步] 按鈕兩次。
  11. 若同意 Microsoft Software 授權條款，請加以接受，然後選擇 [完成] 按鈕。
  12. 當您收到提示時，重新啟動 Eclipse。

從更新網站封存安裝此外掛程式

    注意
    建議您選用上述標準更新網站機制安裝方法，但這需要連線到網際網路，並會安裝 Microsoft 所提供的最新版外掛程式。 若安裝機器未直接連線到網際網路，或組織指定所有人必須安裝特定版本的外掛程式，接下來的方法可能會更加合適。

  1. 使用此頁面上的連結下載 TFSEclipsePlugin-UpdateSiteArchive-14.0.4.zip。
  2. 開啟 Eclipse。
  3. 在 [說明] 功能表上，選擇 [安裝新軟體]。
  4. [安裝] 對話方塊會隨即出現。
  5. 選擇 [新增] 按鈕。
  6. [新增儲存機制] 對話方塊會隨即出現。
  7. 請為名稱輸入 [本機 Team Explorer Everywhere 更新封存]。
  8. 選擇 [封存] 按鈕。
  9. [儲存機制封存] 對話方塊會隨即出現。
  10. 尋找並選取先前所下載的 "TFSEclipsePlugin-UpdateSiteArchive-14.0.4.zip" 檔案，並以其做為檔名。
  11. 選擇 [開啟] 按鈕。
  12. 選擇 [確定] 按鈕。
  13. 在 [安裝] 對話方塊的功能清單中，選取與 [Eclipse 的 Team Explorer Everywhere 外掛程式] 相對應的核取方塊。
  14. 選擇 [下一步] 按鈕兩次。
  15. 若您同意 Microsoft Software 授權條款，請加以接受，然後選擇 [完成] 按鈕。
  16. 當您收到提示時，重新啟動 Eclipse。

安裝 Team Foundation Server 跨平台命令列用戶端

您可以從 Team Foundation Server 跨平台命令列用戶端執行許多版本控制作業。 您必須先安裝此用戶端，並將您的 Shell 或系統路徑設定成包含安裝資料夾。

安裝 Team Foundation Server 跨平台命令列用戶端

  1. 使用此頁面上的連結下載 TEE-CLC-14.0.4.zip。
  2. 解壓縮包含此用戶端的封存 (TEE-CLC-14.0.4.zip file)。
  3. 將您的 Shell 或系統路徑設定成包含存放封存解壓縮後之內容的資料夾。
  4. 若要驗證用戶端已在運作中，請在命令提示輸入 tf，然後按 [ENTER] 鍵。
若用戶端安裝正確，開頭為「Microsoft Team Explorer Everywhere 命令列用戶端 (14.0.4 版)」的輸出會隨即出現。

如需任何命令的詳細資訊，可以輸入 tf help Command，其中 Command 是取得資訊所屬之命令的名稱。 例如，您可以輸入 tf help checkin。 您也可以輸入 tf command /help 或 tf command -help 來取得說明。

注意
您可以在 Shell 中設定環境變數，以控制命令列用戶端的行為， 例如指定所要使用的 HTTP Proxy、啟用自動快取認證，或控制差異/合併工具。 如需要設定之環境變數的相關資訊，請參閱 MSDN 上的 設定環境變數 主題。
