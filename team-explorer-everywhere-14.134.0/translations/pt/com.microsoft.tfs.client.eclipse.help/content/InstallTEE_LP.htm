Instalando o Plug-in do Team Explorer Everywhere e os Pacotes de Idioma para o Cliente da Linha de Comando

Você pode usar o Pacote de Idiomas do Team Explorer Everywhere para adicionar suporte aos seguintes idiomas no Plug-in do Team Explorer Everywhere e no Cliente de Linha de Comando.

  • Português (Brasil)
  • Chinês (Simplificado)
  • Chinês (Tradicional)
  • Francês
  • Alemão
  • Japonês

Pré-requisitos

Você já deve ter instalado o Team Explorer Everywhere e a versão de idioma desejada do Team Foundation Server para usar este pacote de idiomas.

Incluindo Idiomas Adicionais ao Team Explorer Everywhere

Você pode instalar o plug-in do pacote de idiomas para exibir mensagens e janelas usando o idioma do Eclipse escolhido, caso haja suporte do pacote de idiomas. Se uma mensagem em particular não estiver disponível para o idioma selecionado, a versão padrão em inglês será exibida.

Observação
Você deve instalar o pacote de idiomas do produto Eclipse para seu idioma e configurar o Eclipse para usá-lo. Consulte http://www.eclipse.org/babel/ para obter mais informações sobre como baixar e configurar pacotes de idiomas de produtos para o Eclipse.

Para instalar idiomas adicionais do Team Explorer Everywhere no Eclipse

  1. Abra o Eclipse.
  2. No menu Ajuda, clique em Instalar Novo Software. A caixa de diálogo Instalar é exibida.
  3. Clique em Adicionar. A caixa de diálogo Adicionar Site é exibida.
  4. Em Nome, digite Arquivo morto local do Pacote de Idiomas do TFS.
  5. Clique em Arquivo Morto.
  6. Especifique a localização do arquivo morto e clique em Abrir.
    Selecione o arquivo morto baixado, TFSEclipsePlugin-NL1-UpdateSiteArchive-14.0.4.zip.
  7. Clique em OK.
  8. Na lista de recursos na caixa de diálogo Instalar, selecione a caixa de seleção que corresponde ao Pacote de Idiomas do Team Explorer Everywhere.
  9. Clique em Avançar duas vezes.
  10. Aceite os Termos de Licença para Software Microsoft e clique em Concluir.
  11. Reinicie o Eclipse quando solicitado.

Adicionando outros idiomas ao cliente de linha de comando de plataforma híbrida para Team Foundation Server

Você pode instalar o pacote de idiomas para oferecer suporte a idiomas adicionais ao cliente de linha de comando interplataforma para Team Foundation Server. Variáveis de ambiente específicas de localidade devem ser configuradas para seu idioma e codificação de caracteres desejados e seu programa de terminal deve estar configurado para exibir caracteres nessa codificação. Consulte a página do manual locale(1) e a documentação de seu programa de terminal para obter mais informações.

Para instalar os idiomas adicionais do cliente de linha de comando interplataforma

  1. Descompacte o arquivo morto (TEE-CLC-NL1-14.0.4.zip) que contém os idiomas no mesmo local do cliente de linha de comando para várias plataformas.

Observação
O arquivo morto contém um único arquivo com.microsoft.tfs.client.clc.nl1.jar que é extraído para o diretório /lib dentro da instalação do cliente de linha de comando. Confirme se esse arquivo está no diretório /lib da instalação para verificar se a instalação foi bem-sucedida.
